# Car Configuration

You are building a car showroom website that displays different car models.

A car has the following 6 properties. New properties may be added in the future.

1. **Engine** — can be one of: V1 Petrol, V2 Petrol, V1 Diesel, V2 Diesel. Each engine has its own fuel tank capacity, torque, and mileage.

2. **Wheel Type** — can be one of: Alloy, Steel, Off-road. Each wheel type has its own wheel radius.

3. **Infotainment System** — can be one of: Basic, Premium, None.

4. **Suspension** — can be one of: Standard, Sport, Off-road.

5. **Display** — displays the car and its details along with its price.

6. **Drive** — shows the car driving. Since this is a CLI application, print the appropriate message to the console.

Any combination of the 4 variable properties is valid. A car can have a V1 Petrol engine with Off-road wheels, Sport suspension, and a Premium infotainment system — or any other combination.
